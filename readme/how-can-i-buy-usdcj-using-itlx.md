# How can I buy $CJ using ITLX?

{% stepper %}
{% step %}
### Open Your Web3 Wallet

Use any web3 wallet that supports BNB Smart Chain (BSC):

* ITLX Supper Wallet
* Trust Wallet
* MetaMask
* OKX Wallet
* Bitget Wallet
* TokenPocket

Make sure you are connected to **BNB Smart Chain (BSC)** network.
{% endstep %}

{% step %}
### Go to the DApp Browser / Web3 Browser

Inside your wallet:

* Tap “Browser”, “DApp”, or “Web3” (depending on the wallet you use)
{% endstep %}

{% step %}
### Use the Web Search Bar

In the search field, paste the official GeckoTerminal link:

https://www.geckoterminal.com/bsc/pools/0x832e1e4aFeE57AA9ddd2F8C81A31d0814d9A15A0

This will open the **CJ/WBNB trading chart**.
{% endstep %}

{% step %}
### Click “Trade” or “Swap”

Inside GeckoTerminal:

* Scroll down and tap **“Trade on PancakeSwap”**\
  (It will automatically take you to PancakeSwap with your token address loaded.)

If prompted, choose: **PancakeSwap (v2)**
{% endstep %}

{% step %}
### Connect Your Wallet

Tap **“Connect Wallet”** and select your wallet:

* ITLX Supper Wallet
* Trust Wallet
* MetaMask
* OKX Wallet
* etc.
{% endstep %}

{% step %}
### Choose BNB to Swap

Make sure:

* **From:** BNB
* **To:** CJ Token (your contract auto-filled)

Enter the amount of BNB you want to use.
{% endstep %}

{% step %}
### Confirm the Swap

Tap:

* **Swap**
* **Confirm Swap**

Then confirm the transaction inside your wallet and wait for a few seconds for the transaction to complete.
{% endstep %}

{% step %}
### You Now Have $CJ Coin

If the token doesn’t show in your wallet:

* Tap **Add Coin**
* Paste the contract address:

```
0x9ECEfef5d5B5b0aCF2467393e7b5A6c7c501c6F6
```

* Add it manually
{% endstep %}
{% endstepper %}
